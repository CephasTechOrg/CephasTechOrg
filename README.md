## 🚀 Work & Projects (selected)

> Each project includes *purpose*, *stack*, *APIs / integrations*, and *how to demo/test*.

### Vitalgo — Health-Tech Platform (Founder)

* **Purpose:** Real-time vitals capture, predictive alerts, dashboards for clinicians and patients.
* **Stack:** FastAPI (backend), PostgreSQL, SQLAlchemy, React + Tailwind (dashboard), MQTT for device telemetry, Docker, Kubernetes-ready.
* **APIs & integrations:** Stripe (billing & subscriptions), Twilio (SMS alerts), Firebase / Auth0 (auth), WakaTime (developer metrics), Prometheus + Grafana (monitoring), Sentry (errors), Google Cloud Pub/Sub or AWS SNS for eventing.
* **Demo:** `npm run start` (frontend), `uvicorn app.main:app --reload` (backend). Example `.env.example` included in repo.

### Open Source: Vitals‑SDK (Python)

* **Purpose:** Simple, secure SDK for ingesting wearable vitals and sending to an endpoint.
* **Stack:** Python 3.10+, pydantic, requests, black-tested, CI with GitHub Actions.
* **Highlights:** Automatic batching, retry/backoff, strong typing, mock harness for unit tests.
